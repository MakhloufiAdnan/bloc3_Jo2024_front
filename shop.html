Informations importantes

La vente des billets pour les Jeux Olympiques de PARIS 2024 débutera en mars 2024. La date limite pour participer au tirage au sort de la première phase de vente des billets est le 15 fevrier. Inscrivez-vous dès maintenant pour accéder aux meilleures places pour vos sports préférés !

  • Une plateforme de billetterie Unique.
  • Accès depuis le monde entier.
  • Biellets 100% numériques.

Comment ça marche

1

Phase 1 : Commande

  • Les billets pour assister aux épreuves sont disponibles seulement sur le site officiel des Jeux Olympique.
  • Les biellets sont transmis via votre adresse mail.
  • Vous devez vous inscrir et vous connecter pour valider votre panier.
2

Phase 2 : Le jour de l'épreuve.

  • Il convient de presenter le QR code.
  • Une pièces d’identitée peut vous être demandée.
  • Toutes les informations liées à l’épreuve sont transmises par mail.
3

Phase 3 : Après l’épreuve

  • Une enquête de satisfation vous sera transmise, aidez nous à rendre ces jeux inoubliables.
  • Votre QR code vous permettra d’obtenir une remise de 10% dans notre boutique.
